# React Native CLI (no EXPO)

**Для запуска использовать:**

```bash
`$ npm run android`
# или
`$ yarn run android`
# или
`$ pnpm run android`
# или
`$ bun run android`
```

**Используются технологии:**

- TypeScript
- Redux-toolkit
- React Native Paper
- ReactNavigation
- ReactNavigation Native Stack
